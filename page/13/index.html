“超五类”指的是超五类非屏蔽双绞线(UTP—Unshielded Twisted Pair).
超五类线的标识是“CAT5E”，带宽155M;
超五类双绞线主要用于千兆网上，但现在也普通应用于局域网中;
超五类可以***通信距离为300M。并且外面的保护层使用寿命比普通五类线长很多。多数的从小区网关或者整楼网关中连接到你的路由器或者交换器的线就是超五类线。

Javascript基本类型与引用类型

发表于 2020-05-26 更新于 2021-05-09 分类于 Code
  1. 基本类型值在内存中占据固定大小的空间，因此被保存在栈内存中。
  2. 从一个变量向另一个变量复制基本类型的值，会创建这个值的一个副本。
  3. 引用类型的值是对象，保存在堆内存中。
  4. 包含引用类型的变量实际上包含的并不是对象本身，而是一个指向该对象的指针。
  5. 从一个变量向另一个变量复制引用类型的值，复制的其实是指针，因此两个变量最终都指向同一个对象。
  6. 确定一个值是哪种基本类型可以使用typeof操作符，而确定一个值是哪种引用类型可以使用instanceof操作符。

websocket四种状态

发表于 2020-05-21 更新于 2021-05-09 分类于 Code

通常在实例化一个websocket对象之后，客户端就会与服务器进行连接。但是连接的状态是不确定的，于是用readyState属性来进行标识。它有四个值，分别对应不同的状态：

  • CONNECTING：值为0，表示正在连接；
  • OPEN：值为1，表示连接成功，可以通信了；
  • CLOSING：值为2，表示连接正在关闭；
  • CLOSED：值为3，表示连接已经关闭，或者打开连接失败。
